#at一次性计划任务

  • 安装at
    yum install at -y

  • 启动at服务
    systemctl restart atd

  • 设置at一次性计划任务

####举个栗子：

10:05的时候再tmp下创建一个ken文件
[root@localhost:~] at 10:05
at> touch /tmp/ken
at> ＜EOT＞         #按ctrl d进行保存
job 2 at Mon May 13 10:05:00 2019

查看计划任务的两种方式：
at -l或者atq

再次查看计划任务发现计划任务为空
at一次性计划任务执行完成之后自动退出

删除计划任务：

[root@localhost:~]atq
5 Mon May 13 10:15:00 2019 a root
6 Mon May 13 10:15:00 2019 a root

atrm 5 删除计划任务，加上任务前面的序列号

#crontab周期性计划任务

  • 第一步：检查crond是否已经启动
    [root@ken ~]# ps aux | grep “crond”
    root 631 0.0 0.1 126280 1592 ? Ss 08:24 0:00 /usr/sbin/crond -n

  • 第二步：启动crond
    systemctl restart crond

  • 第三步：查看计划任务
    crontab -l

###再举一些栗子：打印时间至costa

  • 监控ken文件

    • [root@ken ~]# tail -f /tmp/ken

  • 编写计划任务

    • [root@ken ~]# crontab -e
      * * * * * /usr/bin/echo 123 >> /tmp/ken

  • 保存
    就是使用vim打开了一个文件，正常wq保存即可

  • 查看计划任务

    • [root@ken ~]# crontab -l
      * * * * * /usr/bin/echo 123 >> /tmp/ken

  • 查看执行结果

    • [root@ken ~]# tail -f /tmp/ken
      123
      123
  • 删除计划任务：
    • 注释掉不想执行的计划任务

[root@ken ~]# crontab -e

          • /usr/bin/echo 123 >> /tmp/ken
          • /usr/bin/echo date “+%F \%T” >> /tmp/ken
            [root@ken ~]# crontab -r
            [root@ken ~]# crontab -l
            no crontab for root

[root@ken ~]# vim /etc/crontab
SHELL=/bin/bash #指定shell类型
PATH=/sbin:/bin:/usr/sbin:/usr/bin #crontab命令执行路径
MAILTO=root #如果有错会发送邮件到/var/spool/mail/root

For details see man 4 crontabs

Example of job definition:

.—————- minute (0 – 59)

| .————- hour (0 – 23)

| | .———- day of month (1 – 31)

| | | .——- month (1 – 12) OR jan,feb,mar,apr …

| | | | .—- day of week (0 – 6) (Sunday=0 or 7) OR sun,mon,tue,wed,thu,fri,sat

| | | | |

* user-name command to be executed

`

#习题：

  1. 每月1、10、22日的4:45重启network服务
    2：每周六、周日的1:10重启network服务
    3：每天18:00至23:00之间每隔30分钟重启network服务
    4：每隔两天的上午8点到11点的第3和第15分钟执行一次重启
    5：每周日凌晨2点30分，运行cp命令对/etc/fstab文件进行备份，存储位置为/backup/fstab-YYYY-MM-DD-hh-mm-ss；
    6：一月一号的4点重启smb
    7：晚上11点到早上7点之间，每隔一小时重启smb
    8：每一小时重启smb

#答案：
1.45 4 1,10,22 /usr/bin/systemctl restart networ
2.10 1 6,7 /usr/bin/systemctl restart network
3./30 18-23 /usr/bin/systemctl restart network
4.3,15 8-11 /2 /usr/sbin/reboot
5.30 2 7 /usr/bin/cp /etc/fstab /tmp/fstab-date “+\%F_\%T”
6.正确答案; 0 4 1 1 /usr/bin/systemctl restart smb #4点钟只执行一次命令
错误答案： 4 1 1 /usr/bin/systemctl restart smb #四点钟执行了60次命令
7.0 23-0/1,1-7/1 /usr/bin/systemctl restart smb
0 23,1,3,5,7 /usr/bin/systemctl restart smb
8.0 /usr/bin/systemctl restart smb
