#vim简单操作指南

  1. vim编辑器
    三种模式：
    命令模式（普通模式）
输入模式（编辑模式）
末行模式
  2. 模式之间的切换
    命令模式–>输入模式（a,i,o）
    输入模式–>命令模式（esc）
    命令模式–>末行模式（：）
    末行模式–>命令模式（esc）

  3. 命令模式下的常用操作
    yy: 复制，nyy就是复制n行
    p: 粘贴
    dd: 剪切
    dG: 删除光标所在行到末尾的全部行
    u: 撤销（等于Windows的ctrl+z）
    ctrl+r：恢复（等于Windows的ctrl+y）
    G：跳转到最后一行
    nG： 跳转到第n行，如1G就是跳转到首行
    home: 跳转到行首
    end: 跳转到行尾
    pgup: 上翻一页,或者ctrl+f
    pgdn: 下翻一页，或者ctrl+b
    /: 查找

    n: 从上往下查找
N：从下往上查找

  4. 末行模式下的操作
    q: 退出
    w: 保存
    wq: 保存并退出
    q!: 强制退出
    wq!: 保存并强制退出

    跳转至n行：冒号 行号
    缩进：
    set ai：自动缩进
    set noai：不自动缩进
    set tabstop=n：设置缩进，tab为n个空格

    r /文件路径/文件a ：导入文件a的内容到当前的文件中
    w /路径/文件名 ：将当前文件另存为

    set nu: 显示行号
    set nonu： 不显示行号
    nohl: 取消高亮
    set hls:打开高亮

    1,3 d: 删除1-3行内容
    1，3 m 5: 移动1,3内容至5行下
    1,3 co 5: 复制1,3内容至5行下

    替换：
    格式：行号起,行号止 s/源字符/替换内容/g (g是global的意思，表示全部替换)
    如果不加g每行只有符合条件的第一段文字会被替换
    其中，/可以被各种符号替代，只要统一就行，如：
    将文件中全部的/替换为+
    :% s@/@+@g
    全局替换：
    1，$ s///g或者% s///g
    示例：
    % s/^/#/g: 在每行的行首添加#
    % s/$/#/g: 在每行的行尾添加#
    % s/^#//g：删除行首的#
    % s/#$//g: 删除行尾的#

    ^：表示行首
    $：表示行尾

  5. vim配置文件
    全局配置：/etc/vimrc
    个人配置：~/.vimrc(默认不存在，需要自己手动创建)

  6. 崩溃缓存文件
    遇到意外情况时，vim会保存一个.原文件名.swp的文件
    在下次编辑此文件时，会提示你是否需要恢复备份
    ※恢复：可以 按R键 查看备份文件，wq保存，再用rm命令删除备份文件，或者使用 vim -r 原文件名 查看备份文件
    ※不想恢复：可以按Q键先退出编辑，用rm命令删除备份文件即可（命令为： rm -f .文件名.swp ）

  7. 分屏操作
    vim -o： 水平分屏
    vim -O: 垂直分屏
    ctrl+w: 切换文件
